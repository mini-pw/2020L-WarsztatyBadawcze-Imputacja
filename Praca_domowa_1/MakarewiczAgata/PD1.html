Trenowanie modelu

Zastosujemy algorytm wyszukiwania k najbliszych ssiadw (kNN). Przewiduje on klas nowej obserwacji na podstawie k najbliszych obserwacji z prby uczcej. Stosowana jest regua wikszociowa.

Zdefiniowanie rodzaju problemu (task)

# zamieniamy zmienne typu factor
titanic$sex <- as.numeric(as.factor(titanic$sex))
titanic$embarked <- as.numeric(as.factor(titanic$embarked))

# nie bdziemy korzystali z "name" oraz "ticket" - s to zmienne typu "char", unikalne dla kadego pasaera 
# na potrzeby modelu musimy rwniez usun "home.dest"
titanic <- titanic%>%
  select(-(name), -(ticket),-(home.dest))

# traget - kolumna, ktora objasniamy
# TaskClassif do problemu klasyfikacji
task_titanic = TaskClassif$new(id = "titanic", backend = titanic, target = "survived")

# podsumowanie zbioru danych
print(task_titanic)
## <TaskClassif:titanic> (1309 x 8)

Podzia na zbir testowy i treningowy

Do zbioru treningowego bierzemy losowe 80% obserwacji z naszego zbioru danych. Zbir testowy bd stanowiy pozostae obserwacje (20%).

# definiujemy, ktre wiersze nale do zbioru treningowego, a ktre do testowego
train_set = sample(task_titanic$nrow, 0.8 * task_titanic$nrow) 
test_set = setdiff(seq_len(task_titanic$nrow), train_set)

data_train <- titanic[train_set,]
data_test <- titanic[test_set,]

Imputacja - metoda 1

Braki danych w kolumnach age oraz fare (jeli si pojawi) uzupenimy redni, w kolumnie embarked - mod.

data_train1 <- data_train

# miss_var_summary(data_train1)

imp <- mice(data_train1%>%select(-(embarked)), method = "mean", m = 1, maxit = 1)

data_train1$embarked <- with(data_train1[,-2], impute(embarked, dominant))

data_train1[,-10] <- complete(imp)

# miss_var_summary(data_train1)

# ---------------------------------------------------------------------------------------

data_test1 <- data_test

# miss_var_summary(data_test1)

imp <- mice(data_test1%>%select(-(embarked)), method = "mean", m = 1, maxit = 1)

data_test1$embarked <- with(data_test1[,-2], impute(embarked, dominant))

data_test1[,-10] <- complete(imp)

# miss_var_summary(data_test1)

Model:

knn_classifier <- knn(train=data_train1[, -2], 
                      cl=data_train1$survived,
                      test=data_test1[, -2], 
                      k=10,

Podsumowanie

W przypadku naszych danych dla zastosowanych metod imputacji danych accuracy oscyluje pomidzy wartociami 0.6 - 0.8 wic jest to cakiem przyzwoity wynik. Najlepiej sprawdzia si imputacja poprzez usunicie wszystkich kolumn z brakami danych - 0.77 accuracy - natomiast najgorzej - imputacja poprzez usunicie obserwacji z brakami danych - 0.6 accuracy. Pomidzy uzupenianiem redni a median nie ma zbyt duej rnicy.
